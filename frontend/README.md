# Lloyd's Delivery - Food Delivery Platform

A full-stack food delivery marketplace web application similar to Uber Eats / Mr D Food.

## 🚀 Features

- **Customer Portal**: Browse restaurants, add to cart, place orders, track deliveries
- **Driver Portal**: Accept deliveries, update status, view earnings
- **Admin Dashboard**: Manage restaurants, users, orders, and analytics
- **Real-time tracking**: Live order status updates
- **Responsive design**: Mobile-friendly interface

## 🛠️ Tech Stack

- **Frontend**: React.js with Vite, TailwindCSS
- **Backend**: Node.js with Express.js (in development)
- **Database**: MySQL (in development)
- **State Management**: React Query + Context API
- **Real-time**: Socket.io (planned)
- **Payments**: Paystack integration (planned)
